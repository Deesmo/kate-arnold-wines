# Kate Arnold Wines Mobile App

A mobile-first PWA (Progressive Web App) for Kate Arnold Wines — designed to be saved to the iPhone home screen.

## Features
- **Age Gate** — 21+ entry check
- **Hero Photo Slider** — auto-plays through real Kate Arnold Wines photography with touch swipe
- **Wine Portfolio** — all 11 wines with real bottle shots, tasting notes, accolades, label images
- **Photo Gallery** — tap-to-expand lightbox with all Squarespace CDN photos
- **About Page** — photo scroll row, story, founder bios, philosophy pillars
- **Find Our Wines** — links to shop, retailer map, wine club, contact
- **Wine Detail Modal** — full-screen bottom sheet with bottle shot, tasting notes, labels, accolades

## Deploy to Render
```
1. Unzip to a folder
2. cd kate-arnold-wines
3. git init
4. git add .
5. git commit -m "init"
6. git remote add origin https://github.com/Deesmo/kate-arnold-wines.git
7. git push -u origin main
```
Then connect repo to Render as a Static Site.

## Add to iPhone Home Screen
1. Open the Render URL in Safari
2. Tap the Share button
3. Select "Add to Home Screen"
4. Name it "Kate Arnold Wines"
